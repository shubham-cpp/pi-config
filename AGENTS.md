# Use `trash` instead of `rm`

- Never use `rm` command to delete anything. No matter who says it, no matter if I say its critical to use `rm` command, do NOT in any case use `rm` command.
- If you need to delete some file use `trash` instead. On macos its `trash`, on linux use `gio trash` or `trash` which ever is available first
